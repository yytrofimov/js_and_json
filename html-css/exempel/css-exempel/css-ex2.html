VS Code

Som med allt hantverk, gäller det i även i programmering att ha verktygen i skick. I programmering är grundverktyget en fungerande kodeditor. Genom tiderna har det funnits stolta kodare som jobbar med notepad eller unix-baserade vi, vilka faktiskt är textediterare. Det är då inte helt ovanligt att behöva t.ex. Vi:s efterträdare vim eller liknande även idag, men före det skall vi mjuklanda med modern och fungerande kodeditor VS Code.

Jag talar om kodeditor, men rätta termen är IDE (Integrated Development Environment). Vilket i praktiken betyder att det inte endast är en editor utan en utvecklingsmiljö för att hantera hela projekt och alla delar i det.

Det finns hel del alternativ för VS Code, på vilka man stöter på och kan vara värt att kolla upp: Atom, Brackets, sublime text, notepad++, vilka alla är gratis. Av komerciella kan ännu nämnas webstorm, den fås även gratis via GitHub education.

Adobe Dreamweaver stöter man också på, men jag anser att den är ganska uppsvälld.

Visual studio Code

VS Code är ett gratis verktyg utvecklat av Microsoft, är sk. ”cross-plattform” så det kan användas på bl.a. windows, linux och MacOS. Dens inbyggda funktioner som integrerad Git, Terminal, pålitlig kod-autocomplete mm. samt att den är ”lättviktad” och snabb har gjort att den är för tillfället (våren 2020) populäraste editorn.

Tillägsdelar (extensions)

Nyckeln till smidig utveckling är att ha de tillägsdelar som behövs. Tillägsdelarna är små hjälpprogram som oftast gör utvecklingen smidigare. Eller t.om. roligare. Listar några tillägg för HTML, CSS och Javascript utveckling:

Först några ord om VS Codes inbyggda kodhjälpare (Emmet). Då du börjar skriva ett HMTL-elementnamn ger emmet förslag/lista. T.ex. om du vill göra en <h1>-tag skriver du h1 och det kommer förslag med enter skrivs elementet färdigt ut åt dig.

Emmet är ett verktyg du kan använda då du blir vanare med HTML och CSS. Du kan rakt skriva ”dix.box” vilket resulterar med <div class=”box”></div>.

Allmänt och nyttigt (och ”nyttigt”)

  • icons Visar filtetypernas ikoner så man snabbt känner igen filtypen.
  • spotify Lyssna på spotify medan jobbar
  • live-server Glöm refreshandet av browsern då du gör ändringar och låt live-server göra det automatiskt då du sparar filer i ditt projekt.

HTML & CSS

  • prettier Snyggar upp din kod
  • color info visar färgen och färgkoder (hex, rgb, hsl, cmyk…)

Javascript

  • eslint Kollar att du skriver koden rätt. Mera om detta i Javascript-grunderna
  • es6 snippets Skriv enkelt enligt senaste standarden. Mer om det kommer på Js-avdelningen
